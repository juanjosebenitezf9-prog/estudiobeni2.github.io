Control de salidas tempranas — Proyecto para colegios

Resumen del proyecto

Esta página es la maqueta de un sistema para que los colegios lleven registro claro y ordenado de los estudiantes que salen antes de la hora de salida. Permite registrar quién salió, curso, fecha, hora y motivo, generando un historial que facilita el seguimiento y la comunicación con familias.

Guía rápida

Objetivo: reducir la desorganización y mejorar la trazabilidad de salidas tempranas.

  1. Registrar al alumno con nombre, curso y documento.
  2. Anotar motivo y adjuntar autorización (si aplica).
  3. Generar reporte semanal y exportarlo para la dirección.
Tecnologías sugeridas:
Frontend: HTML/CSS/JS — Backend: PHP, Node.js o Python — DB: MySQL / PostgreSQL
